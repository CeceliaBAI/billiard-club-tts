# 音频文件目录

将预制音频文件放入此目录。文件名需与 config.json 中 `file` 字段对应：

| 文件名 | 播报内容 |
|--------|----------|
| welcome.m4a | 欢迎光临六六台球，祝您击球精准，玩得愉快。 |
| no_smoking.m4a | 请勿将烟头扔在地毯上，请将烟头扔在灭烟水盆中，感谢您的配合。 |
| add_time.m4a | 关灯后请联系前台加时，请勿黑灯打球，感谢您的配合。 |
| reminder.m4a | 离店前请您保管好随身物品 |

如需增减按钮，编辑 config.json 的 `buttons` 列表，并将对应音频文件放入此目录即可。
